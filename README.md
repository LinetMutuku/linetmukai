# Linet Mukai - Portfolio

My personal portfolio website showcasing my projects, skills, and experience as a Frontend Developer.

## Live Site

**[https://linetmukai.vercel.app/](https://linetmukai.vercel.app/)**

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Features

- Responsive design
- Animated UI components
- Project showcase
- Skills section
- Contact form

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
